# La Villa Imports - B2B Field Invoicing App

A mobile-friendly invoicing application for field sales agents to generate e-signable B2B invoices.

## Features

- WooCommerce integration for B2B customer and product data
- Real-time invoice generation
- E-signature capability
- Offline-capable for field use

## Tech Stack

- **Backend**: Node.js, Express
- **Frontend**: HTML, CSS (BEM), Vanilla JavaScript
- **Integration**: WooCommerce REST API
- **Authentication**: JWT

## Setup Instructions

### Prerequisites

- Node.js (v14 or higher)
- WooCommerce store with REST API enabled
- WooCommerce API credentials

### Installation

1. Clone the repository:

```bash
git clone https://github.com/Hambonoire/La-Villa-Imports-B2B-Field-Invoicing-App.git
cd La-Villa-Imports-B2B-Field-Invoicing-App
```
